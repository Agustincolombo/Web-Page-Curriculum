# Portfolio Web - Agustín Colombo

Este repositorio contiene el código fuente de mi **Portafolio Personal y Currículum Web**. Es una Single Page Application (SPA) desarrollada con **Angular 18**, diseñada para presentar mi experiencia profesional, habilidades técnicas y proyectos destacados de manera estructurada y responsiva.

## Descripción

El proyecto consiste en una aplicación web moderna que sirve como carta de presentación profesional. Su objetivo es ofrecer una interfaz limpia, accesible y estéticamente agradable que refleje mis habilidades técnicas como desarrollador, permitiendo acceder fácilmente a mi información profesional y portafolio de trabajos.

## Características

- **Diseño Responsivo**: Interfaz totalmente adaptable que garantiza una visualización óptima en dispositivos móviles, tablets y monitores de alta resolución.
- **Estilo Personalizado**: Implementación de un sistema de diseño propio utilizando una paleta de colores coherente y tipografía moderna (Montserrat), gestionado a través de variables CSS y preprocesador SASS.
- **Arquitectura Modular**: Estructura basada en componentes reutilizables de Angular para facilitar el mantenimiento y la escalabilidad del código.
- **Navegación Intuitiva**: Implementación de navegación fluida y jerarquía visual clara para mejorar la experiencia del usuario.

## Tecnologías Utilizadas

Este proyecto ha sido construido utilizando las siguientes tecnologías y herramientas:

- **Framework**: Angular v18
- **Lenguaje Principal**: TypeScript
- **Estilos y Diseño**: SASS (.sass), CSS Variables, Bootstrap 5 (Grid System)
- **Fuentes**: Google Fonts (Montserrat)
- **Control de Versiones**: Git
- **Despliegue**: Vercel

## Instalación y Ejecución Local

Para ejecutar este proyecto en su entorno local, siga los siguientes pasos:

1.  **Clonar el repositorio**
    Clone el proyecto en su directorio local utilizando el siguiente comando:
    ```bash
    git clone https://github.com/Agustincolombo/wb-curriculum.git
    cd wb-curriculum
    ```

2.  **Instalar dependencias**
    Asegúrese de tener Node.js y npm instalados en su sistema. Ejecute el siguiente comando para instalar todas las dependencias necesarias:
    ```bash
    npm install
    ```

3.  **Ejecutar servidor de desarrollo**
    Inicie el servidor local con el comando:
    ```bash
    ng serve
    ```
    Una vez iniciado, abra su navegador y navegue a `http://localhost:4200/`. La aplicación se recargará automáticamente ante cualquier cambio en los archivos fuente.

## Construcción (Build)

Para generar una versión optimizada para producción, ejecute:

```bash
ng build
```

Los archivos resultantes se generarán en el directorio `dist/`. Estos archivos están minificados y optimizados, listos para ser desplegados en cualquier servidor web.
